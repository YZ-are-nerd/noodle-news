# noodle-news
Простенькое, но довольное мощное приложение для просмотра новостей.
- В основе React
- TailwindCSS
- MobX
- Framer-Motion
- Luxon
- React-Icons
- React-Use
- TypeScript
## В версии 1.0
Будет реализовано просмотр новостей по топикам (темам)
## В версии 2.0
Добавиться поиск по заголовкам новостей
